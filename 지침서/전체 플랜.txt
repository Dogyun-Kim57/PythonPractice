전체 플랜 로드맵 (학습 + 구현 순서)
-------------------------------------------------------------------
0단계. “프로젝트가 돌아가는 최소 상태” 만들기

목표: 서버 켜지고, 라우팅 1개가 동작한다

배우는 것: create_app, blueprint 등록, 템플릿 렌더링 기본
-------------------------------------------------------------------
1단계. DB 연결/쿼리 기본기

목표: MySQL 연결 + 간단 SELECT 한 번 성공

배우는 것: pymysql 커넥션, cursor, commit/rollback, DictCursor
-------------------------------------------------------------------
2단계. 회원가입

목표: members 테이블에 insert

배우는 것: POST 폼 처리, 유효성 검사, UNIQUE 중복 처리, 비밀번호 해시
-------------------------------------------------------------------
3단계. 로그인/세션

목표: 로그인 성공 시 session에 user 저장, 로그아웃

배우는 것: session, flash 메시지, 로그인 상태 확인
-------------------------------------------------------------------
4단계. 게시글 목록/상세

목표: posts list/detail 화면

배우는 것: JOIN(작성자 표시), 템플릿에서 반복문, URL 파라미터
-------------------------------------------------------------------
5단계. 글 작성 + 파일 업로드

목표: 글 저장 + 파일 저장 + attachments insert

배우는 것: multipart/form-data, request.files, secure_filename, uuid 저장명
-------------------------------------------------------------------
6단계. 글 수정 “본인만”

목표: user/manager는 본인글만 수정 가능

배우는 것: owner 체크(get_owner_id), decorator로 권한 분리
-------------------------------------------------------------------
7단계. 삭제 권한 “admin은 전부, 나머지는 본인만”

목표: admin 전부 삭제 가능, user/manager는 본인만

배우는 것: role 기반 권한 정책
-------------------------------------------------------------------
8단계. 다운로드/보안 기본

목표: 첨부파일 다운로드, 경로 안전

배우는 것: send_file, 파일 경로 관리, 업로드 폴더 분리
-------------------------------------------------------------------
